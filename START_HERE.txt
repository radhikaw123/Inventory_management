================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: "npm is not recognized"
Solution: Install Node.js and restart computer

Problem: "Cannot find module 'express'"
Solution: Run "npm install" again

Problem: "Port 3000 already in use"
Solution: Close other applications using port 3000

Problem: Still seeing "Error loading products"
Solution: Make sure server is running (npm run test) and you're using 
          http://localhost:3000 not file://
